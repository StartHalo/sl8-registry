# AI Video Generation Examples

Common use cases and example commands for the ai-video CLI.

## Basic Usage

### Simple Video Generation

```bash
ai-video generate \
  --prompt "A cinematic drone shot flying over mountains at sunset" \
  --model "bytedance/seedance-1-pro-fast"
```

### Using Configuration File

```bash
ai-video generate \
  --prompt "Ocean waves crashing on a beach" \
  --model "google/veo-3.1-fast" \
  --config .ai-kits.json
```

## Advanced Techniques

### Using Seeds for Reproducibility

Generate consistent video styles:

```bash
ai-video generate \
  --prompt "Cyberpunk city street at night with neon signs" \
  --model "bytedance/seedance-1-pro-fast" \
  --duration 10 \
  --seed 42
```

Re-use the same seed to get consistent visual style:

```bash
# Generate multiple clips with same visual style
ai-video generate --prompt "Scene 1: ..." --model "..." --seed 42 --duration 5
ai-video generate --prompt "Scene 2: ..." --model "..." --seed 42 --duration 5
ai-video generate --prompt "Scene 3: ..." --model "..." --seed 42 --duration 5
```

### Cost-Effective Long Videos

Use MiniMax for longer videos (tiered pricing benefits):

```bash
# Better pricing for 11+ seconds
ai-video generate \
  --prompt "Documentary-style footage of coral reef ecosystem, underwater camera" \
  --model "minimax/hailuo-2.3" \
  --duration 20 \
  --resolution 1080p
```

### Resolution Trade-offs

Test with lower resolution, then generate final in high resolution:

```bash
# Test iteration (faster, cheaper)
ai-video generate \
  --prompt "Your test prompt" \
  --model "bytedance/seedance-1-pro-fast" \
  --duration 5 \
  --resolution 480p

# Final production (once satisfied with prompt)
ai-video generate \
  --prompt "Your refined prompt" \
  --model "bytedance/seedance-1-pro" \
  --duration 10 \
  --resolution 2160p
```

## Checking Costs Before Generation

### Estimate Command

Always check costs before generating expensive videos:

```bash
# Estimate single video
ai-video estimate \
  --model "google/veo-3.1" \
  --duration 15 \
  --resolution 1440p

# Estimate with different models for comparison
ai-video estimate --model "bytedance/seedance-1-pro-fast" --duration 10
ai-video estimate --model "google/veo-3.1" --duration 10
ai-video estimate --model "minimax/hailuo-2.3" --duration 20
```

### Check Balance

Verify you have sufficient credits:

```bash
ai-video balance

# JSON output for scripting
ai-video balance --output-format json
```

## Common Workflows

### Development Workflow

1. Estimate costs:
```bash
ai-video estimate --model "bytedance/seedance-1-pro-fast" --duration 10
```

2. Generate test video (fast model, low resolution):
```bash
ai-video generate \
  --prompt "Your test prompt" \
  --model "bytedance/seedance-1-pro-fast" \
  --duration 5 \
  --resolution 480p \
  --quiet
```

3. Refine prompt and generate final (high-quality model):
```bash
ai-video generate \
  --prompt "Your refined prompt" \
  --model "google/veo-3.1" \
  --duration 10 \
  --resolution 1080p
```

### Batch Processing Workflow

Generate multiple videos with different prompts:

```bash
#!/bin/bash
PROMPTS=(
  "Scene 1: Opening shot of sunrise over city"
  "Scene 2: People walking in busy marketplace"
  "Scene 3: Close-up of food being prepared"
)

for prompt in "${PROMPTS[@]}"; do
  ai-video generate \
    --prompt "$prompt" \
    --model "bytedance/seedance-1-pro-fast" \
    --duration 5 \
    --output-format json >> results.jsonl
done
```

### Multi-Resolution Workflow

Generate the same video in multiple resolutions for different platforms:

```bash
PROMPT="Your video prompt here"
MODEL="bytedance/seedance-1-pro"

# Desktop/YouTube (1080p widescreen)
ai-video generate --prompt "$PROMPT" --model "$MODEL" --duration 10 --resolution 1080p --aspect-ratio 16:9

# Mobile/Instagram Stories (vertical)
ai-video generate --prompt "$PROMPT" --model "$MODEL" --duration 10 --resolution 1080p --aspect-ratio 9:16

# Instagram Feed (square)
ai-video generate --prompt "$PROMPT" --model "$MODEL" --duration 10 --resolution 1080p --aspect-ratio 1:1
```

## Troubleshooting Common Issues

### Insufficient Credits

**Error**: "Insufficient credits"

**Solution**:
```bash
# Check balance
ai-video balance

# Estimate cost
ai-video estimate --model "..." --duration X

# Use cheaper model or shorter duration
ai-video generate --prompt "..." --model "bytedance/seedance-1-pro-fast" --duration 5
```

### Model Not Supported

**Error**: "Model not supported" or "Model doesn't support video generation"

**Solution**:
- Verify model name format: `provider/model`
- Check spelling of model name
- Consult `references/models.md` for supported models
- Ensure you're using a video model (not an image model)

### Generation Too Long

**Error**: "Duration exceeds maximum"

**Solution**:
- Maximum duration is 30 seconds per generation
- Split longer videos into multiple generations
- Use seed to maintain visual consistency across clips

### Authentication Failed

**Error**: "Authentication failed"

**Solution**:
```bash
# Verify configuration
cat .ai-kits.json

# All video models require Replicate token
# Check that providers.replicate.apiToken is set correctly
```

## Pro Tips

1. **Start Fast, Finish Slow**: Use fast/cheap models for iteration, premium models for finals

2. **Seed Consistency**: Use seeds to maintain visual style across multiple clips

3. **Duration Sweet Spots**:
   - ByteDance: Any duration works well
   - MiniMax: 11+ seconds gets best per-second pricing
   - Veo: 10-15 seconds balances quality and cost

4. **Resolution Strategy**: Test at 480p/720p, produce at 1080p/1440p

5. **Aspect Ratio Planning**: Know your target platform before generating

6. **Batch Operations**: Generate multiple clips with consistent settings for efficiency

7. **Cost Monitoring**: Always estimate before bulk operations

8. **Prompt Engineering**: Detailed, specific prompts with camera angles and lighting details produce better results

9. **Audio Consideration**: Only Veo models support audio - factor the flat audio cost into budget

10. **Model Selection**: Match model to use case (fast for testing, pro for production)
