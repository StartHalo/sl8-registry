# Style — stickman channel

This file defines the **visual style** for your stickman channel (Layer 2 seed). It is
read at the start of every episode and is the source of the frozen STYLE / DISCIPLINE /
CONSTRAINTS blocks in `seed.manifest.json`. Edit it to change the look, then run
**Update Character** (reset) so the change takes effect and the anchors regenerate.

---

## Style stack
Hand-drawn pencil sketch animation style, visible graphite grain, subtle smudging,
light cross-hatching, varied line weight, on plain white paper background.

## Discipline
Environments and objects rendered with realistic structure, weight, and light pencil
shading. Communicate narrative through posture and spatial composition alone; no
exaggerated facial expressions. Consistent lighting direction. The figure stays minimal
while the world stays believable.

## Positive constraints
Monochrome graphite on white paper only. Single figure, one readable action per frame.
Minimal sparse environment.

## Video style lock
(First line of every animation prompt — do not change unless you change the whole style)
A stick figure hand-drawn pencil sketch animation.

## Audio directive
NO MUSIC, ONLY AMBIENT SOUND. NO TALKING.
